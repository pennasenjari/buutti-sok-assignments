Assignment 16.7: Brick Wall

Create a new React App, and remove the unnecessary parts.

Create a component Brick. It should have a div that is roughly brick shaped and reddish brown.
Create a component Row. It should have ten bricks in a row. 
Create a component Wall. It should have ten rows, making a brick wall.

Extra: Modify your code so that uneven rows are offset by 50% of the brick width.
